Minimize optimization objective over constraints.

Parameters

  • x0 (NDArray) – Initial guess for the solution.

  • xtol (float) – Tolerance for the differences in x, by default 1e-8.

  • gtol (float) – Tolerance for the KKT system, by default 1e-8.

  • mtol (float) – Tolerance for the log barrier parameter mu, by default 1e-6.

  • max_iter (int) – Maximum number of iterations, by default 100.

  • mu (float) – Initial interior point bairrier parameter, by default 1.0.

  • update_mu_every (int) – Parameter mu updating frequency, by default 5.

  • scale_mu (float) – Shrinkage factor for mu updates, by default 0.1

  • verbose (bool) – Indicator of if print out convergence history, by default False

Returns

The result of the solver.
