minimize(x0, xtol=1e-08, gtol=1e-08, mtol=1e-06, max_iter=100, m_init=1.0, m_freq=5, m_scale=0.5, a_init=1.0, a_const=0.01, a_scale=0.9, a_lb=0.001, verbose=False, mat_solve_method='', mat_solve_options=None)[source]#

Minimize optimization objective over constraints.

Parameters:

  • x0 (NDArray) – Initial guess for the solution.

  • xtol (float) – Tolerance for the differences in x, by default 1e-8.

  • gtol (float) – Tolerance for the KKT system, by default 1e-8.

  • mtol (float) – Tolerance for the log barrier parameter m, by default 1e-6.

  • max_iter (int) – Maximm number of iterations, by default 100.

  • m_init (float) – Initial interior point bairrier parameter, by default 1.0.

  • m_freq (int) – Parameter m updating frequency, by default 5.

  • m_scale (float) – Shrinkage factor for m updates, by default 0.1

  • a_init (float) – Initial step size, by default 1.0.

  • a_const (float) – Constant for the line search condition, the larger the harder, by default 0.01.

  • a_scale (float) – Shrinkage factor for step size, by default 0.9.

  • a_lb (float) – Lower bound of the step size when the step size is below this bound the line search will be terminated.

  • verbose (bool) – Indicator of if print out convergence history, by default False

  • mat_solve_method (str) – Method to solve the linear system, by default “”.

  • mat_solve_options (dict | None) – Options for the linear system solver, by default None.

Returns:

The result of the solver.

Return type:

IPResult
